:::caution[Solid 2.0 — `Index` is removed]
The `<Index>` component has been removed in Solid 2.0. Use `<For keyed={false}>` for non-keyed list iteration where items are keyed by array index.
:::

## Migration

```tsx
// 1.x
<Index each={state.list} fallback={<div>Loading...</div>}>
  {(item, index) => (
    <div>
      #{index} {item()}
    </div>
  )}
</Index>

// 2.0
<For each={state.list} keyed={false} fallback={<div>Loading...</div>}>
  {(item, index) => (
    <div>
      #{index()} {item()}
    </div>
  )}
</For>
```

Note that in Solid 2.0, both `item` and `index` are accessor functions in `<For>` — call `item()` and `index()` to read values.

See the [`<For>` reference](/reference/components/for) for full documentation.
