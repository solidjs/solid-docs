`children` resolves a component's `children` prop and returns an accessor for the resolved result.

## Import

```ts
import { children } from "solid-js";
```

## Type

```ts
function children(fn: Accessor<JSX.Element>): ChildrenReturn;

type ChildrenReturn = Accessor<ResolvedChildren> & {
	toArray: () => ResolvedChildren[];
};
```

## Parameters

### `fn`

- **Type:** `() => JSX.Element`
- **Required:** Yes

Accessor that returns the component's `children` value.

## Return value

- **Type:** `ChildrenReturn`

Returns an accessor for the resolved children.
The accessor also exposes `toArray()`.

## Behavior

- `children` resolves functions, arrays, fragments, and nested child structures.
- The returned accessor reads the current resolved value of `props.children`.
- `toArray()` returns a flattened array of resolved child elements.

## Examples

### Basic usage

```tsx
import { children } from "solid-js";

function Wrapper(props) {
	const resolved = children(() => props.children);

	return <div>{resolved()}</div>;
}
```

### Flatten children into an array

```tsx
import { children } from "solid-js";

function List(props) {
	const resolved = children(() => props.children);
	const items = resolved.toArray();

	return (
		<ul>
			{items.map((child) => (
				<li>{child}</li>
			))}
		</ul>
	);
}
```
