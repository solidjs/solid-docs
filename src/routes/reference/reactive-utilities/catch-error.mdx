:::caution[Solid 2.0 — `catchError` is removed]
`catchError` (and the older `onError`) have been removed in Solid 2.0. Use the [`Errored`](/reference/components/error-boundary) component for error boundaries instead.
:::

## Replacement: `Errored`

```tsx
// 1.x
import { catchError } from "solid-js"

catchError(
  () => riskyOperation(),
  (err) => console.error("Caught:", err)
)

// 2.0
<Errored fallback={(err, reset) => <div>Error: {err.message}</div>}>
  <RiskyComponent />
</Errored>
```

See the [`Errored` reference](/reference/components/error-boundary) for full documentation.
